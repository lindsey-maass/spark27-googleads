# Google Ads Mini Course — Spark27 Creative

An 8-module interactive course for the Spark27 team. Open `index.html` in any browser or host on GitHub Pages.

## Modules
1. How Google Ads Works
2. Account Structure
3. Keyword Strategy
4. Writing Effective Ads
5. Bidding and Budgets
6. Targeting and Audiences
7. Reading Your Results
8. Ongoing Optimization

## File Structure
```
index.html          ← Open this in a browser
docs/
  module-1-*.docx   ← Downloadable reference docs (one per module)
  module-2-*.docx
  ...
```

## Publishing to GitHub Pages
1. Create a new GitHub repo
2. Upload all files (keeping the `docs/` folder intact)
3. Go to Settings → Pages → Source: main branch / root
4. Your course will be live at `https://yourusername.github.io/repo-name`
